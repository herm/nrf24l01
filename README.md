# NRF24L01+ driver
This driver is designed to be used in an environment that provides [MBED](http://mbed.org)-style classes.

# Classes used
- `DigitalOut`
- `SPI`


# Black NRF24L01+ module from Ebay:

   GND (marked)   1   2   VDD
	     CE   3   4   CSN
	    SCK   5   6   MOSI
	   MISO   7   8   IRQ
